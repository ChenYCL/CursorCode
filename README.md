# 主要功能

- 📃智能对话：可以在侧边栏直接与机器人对话
- 🔥代码生成：在代码中输入需求来生成代码
- 📝代码优化：在代码中输入需求对代码进行优化

# 开始使用

在vscode扩展商店中搜索CursorCode

重载窗口，在右侧选中Cursor图标进入

右键编辑器，选择代码进行操作
